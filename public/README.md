# Public Assets Folder

Place your profile picture here.

## Profile Picture Instructions:

1. **Save your profile picture** in this folder (`public/`)
2. **Name it exactly:** `profile.jpg` (or `profile.png` if it's a PNG file)
3. **Recommended size:** 600x600 pixels or larger
4. **Supported formats:** JPG, PNG, WebP

The image will automatically appear in the About section of your portfolio.

## Current Setup:
- The portfolio is configured to look for: `/profile.jpg`
- If the image is not found, a placeholder will be shown temporarily
