# HOW TO ADD YOUR PROFILE PICTURE

## Quick Steps:

1. **Find your profile picture file** on your computer

2. **Copy it** to this location:
   ```
   C:\Users\bilal\OneDrive\Desktop\InternShip Acme-One\Portfolio\public\profile.jpg
   ```

3. **The file MUST be named exactly:** `profile.jpg`

4. **If your image is PNG format**, name it `profile.png` instead

5. **Restart your development server:**
   - Press `Ctrl + C` in the terminal to stop it
   - Run `npm run dev` again

## File Location:
```
Portfolio/
  └── public/
      └── profile.jpg  ← PUT YOUR IMAGE HERE
```

## Supported Formats:
- ✅ `.jpg` or `.jpeg`
- ✅ `.png`
- ✅ `.webp`
